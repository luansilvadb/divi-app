# ROADMAP ESTRATÉGICO: divi
**Mente Brilhante Ω - Modo Visionário Ativado**

## THE CORE TRUTH
As pessoas vivem num abismo de ignorância financeira porque os sistemas atuais são lentos, invasivos e burocráticos. A verdade central é: *Ninguém quer gerenciar dinheiro; eles querem paz de espírito.* Para alcançar a paz, o controle deve ser invisível, absoluto e imediato.

## THE NORTH STAR
O "divi" não será apenas um aplicativo. Será o primeiro cérebro financeiro local-first, onde a privacidade é o alicerce e a performance é mágica. Nossos usuários abrirão o app e, em menos de 100 milissegundos, saberão exatamente o pulso de sua vida financeira — sem a necessidade de depender de servidores em nuvem lentos ou ceder dados corporativos para terceiros.

## THE PARADIGM SHIFTS
1. **Velocidade como Recurso Primário:** Um toque, e os dados devem estar lá. O banco de dados vive no navegador do usuário (Dexie).
2. **Offline é o Novo Online:** Você não precisa de sinal no meio de uma floresta para registrar uma compra.
3. **Privacidade Criptográfica:** Nós fornecemos a estrutura, o usuário mantém as chaves. O servidor (Supabase) só atua quando é estritamente necessário para sincronização e auth.
