# Quickstart: Refine Categories UI Aesthetics

## Running the Application
To verify the aesthetic changes, start the development server:

```bash
npm run dev
```

## Verification Steps
1. Open the application in a web browser.
2. Navigate to the **Categorias** view.
3. Switch the application to **Dark Mode**.
4. Observe the category cards (`.apple-category-card`) — ensure no light-mode shadows are visible and the hover state reflects correctly.
5. Switch the application to **Light Mode**.
6. Observe the category cards — ensure they elevate gracefully on hover with the design system shadow.
7. Observe the icon container — its border-radius should match other `radius-lg` items.
8. Check the sidebar stats panel to ensure it casts a consistent design-system shadow.
