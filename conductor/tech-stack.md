# Tech Stack

## Frontend
- **Framework**: Vue 3 (Composition API)
- **Language**: TypeScript
- **Build Tool**: Vite
- **UI Library**: PrimeVue (with PrimeUIX themes)
- **State Management**: Pinia
- **Routing**: Vue Router
- **Charts**: Chart.js
- **Icons**: PrimeIcons
- **Styling**: Tailwind CSS

## Backend & Data
- **BaaS**: Supabase (Authentication, Database)
- **Local Database**: Dexie.js (IndexedDB wrapper) - supports offline-first/caching strategy.

## Testing
- **Unit**: Vitest
- **E2E**: Playwright

## Linting/Formatting
- **Linting**: ESLint, Oxlint
- **Formatting**: Prettier
