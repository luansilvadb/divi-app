# Tech Stack: divi

## Core Technologies
- **Vue 3:** Reactive and performant frontend framework.
- **Vite:** High-speed build tool and development server.
- **PrimeVue:** Comprehensive UI component library for Vue 3.
- **Supabase:** Backend-as-a-service for authentication, real-time database (PostgreSQL), and storage.
- **Tailwind CSS:** Utility-first CSS framework for rapid and consistent styling.
- **TypeScript:** Type-safe development to catch errors early and improve maintainability.
- **Pinia:** Modern state management for Vue 3.
- **Vue Router:** Official router for Vue.js applications.

## State Management & Persistence
- **Dexie:** Lightweight wrapper for IndexedDB to handle offline data storage.
- **VueUse:** Essential collection of Vue Composition API utilities.

## Data Visualization
- **Chart.js:** Robust and flexible charting library for financial reports.

## Styling & Icons
- **PrimeIcons:** Consistent and comprehensive set of icons for the UI.
- **PostCSS:** Tool for transforming CSS with JavaScript.

## Testing & Quality Assurance
- **Vitest:** Blazing fast unit testing framework.
- **Playwright:** Reliable end-to-end testing for modern web apps.
- **Vue Test Utils:** Official library for testing Vue components.
- **ESLint:** Pluggable linting utility for JavaScript and TypeScript.
- **Prettier:** Opinionated code formatter.
- **Oxlint:** Extremely fast JavaScript linter.

## Build & Deployment
- **Vite Plugin PWA:** Seamless integration for creating Progressive Web Applications.
- **Workbox:** Set of libraries for adding offline support to web apps.
- **npm-run-all2:** Utility to run multiple npm scripts in parallel or sequential order.
- **jiti:** Runtime TypeScript/ESM executor for Node.js.
- **Vercel:** Optimized hosting and deployment platform (indicated by `vercel.json`).

## Architecture
- **Single Page Application (SPA):** Modern web app architecture for a smooth user experience.
- **Progressive Web App (PWA):** Offline support and native-like feel.
- **Module-Based Directory Structure:** Organized features into dedicated modules for scalability (e.g., `src/modules/`).
