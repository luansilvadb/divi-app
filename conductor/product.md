# Product Definition: Divi App

## Overview
Divi is a comprehensive personal finance management application designed to help users track transactions, manage budgets, set financial goals, monitor loans, and analyze spending habits. It provides a holistic view of the user's financial health through an intuitive dashboard and detailed reports.

## Key Features
- **Transactions**: Add, edit, and categorize income and expenses. Advanced search and filter capabilities.
- **Dashboard**: Visual overview of financial status, including charts and summaries.
- **Budgets**: Define and track spending limits per category.
- **Goals**: Set savings goals and track progress.
- **Loans**: Manage money owed to or by the user.
- **Subscriptions**: Track recurring payments.
- **Reports**: Detailed analysis of spending and income over time.
- **Calendar**: View financial events in a calendar format.
- **Auth**: Secure user authentication via Supabase.

## Target Audience
Individuals looking for a robust, privacy-focused tool to manage their personal finances, with support for offline capabilities.

## Design Philosophy
- **Modern & Clean**: Utilizing glassmorphism, smooth animations, and a polished UI.
- **Mobile-First**: Responsive design ensuring usability on all devices.
- **User-Centric**: Intuitive navigation and clear data visualization.
